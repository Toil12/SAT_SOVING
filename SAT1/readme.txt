We use the cadical in PySAT package,cadical as solver use PyQt5 as GUI

Dependencies(not use anaconda):

To install PySAT

$pip install python-sat

To install pyqt5:

$pip install PyQt5

After above run Main.py you could see the auto-make function and read local puzzle file in the path :

SAT1/env

If sometimes break up ,it means that the problem is out of the limitataion of original rules.
